Resumen Ejecutivo del Proyecto:
Este proyecto consiste en la creación de un ecosistema de análisis de datos para una cadena de Retail. El objetivo principal fue transformar una base de datos masiva y desordenada en una herramienta visual de alta precisión que permita optimizar la toma de decisiones comerciales.

Desafíos y Soluciones Técnicas
1. Procesamiento de "Big Data" con Python
El primer gran reto fue manejar un volumen de datos que superaba las capacidades estándar de carga .

Lógica de Limpieza: Utilicé Python para la limpieza de nulos y la corrección de formatos en columnas críticas como amount y product_category.

Eficiencia: Gracias a Pandas y NumPy, logré que la data fuera lo suficientemente ligera y estructurada para que Power BI la procesara sin errores.

2. Modelado y Visualización Estratégica
En Power BI, diseñé un dashboard con un enfoque de UI/UX moderno (Dark Mode) que prioriza la legibilidad de los KPIs financieros.

Resultados de Negocio: El análisis reveló ingresos totales por 402.51 millones, con un liderazgo claro de la categoría Electronics (95M).

Descubrimiento de Patrones: Identifiqué que el 62.07% de los clientes son hombres y detecté una caída crítica en las ventas después de julio de 2023, lo que permite a la empresa planificar mejores promociones para el cierre de año.
